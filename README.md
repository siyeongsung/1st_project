# 1차 프로젝트 자료집

## 프로젝트명
생성형 AI 기반 영양소 패턴 분석을 통한 고콜레스테롤혈증 위험도 예측

## 프로젝트 소개
본 프로젝트는 식생활 변화와 영양 불균형으로 인해 증가하는 고콜레스테롤혈증 위험에 주목하여, 개인의 영양소 섭취 패턴을 분석하고 이를 기반으로 위험도를 예측하는 서비스를 기획한 프로젝트입니다.  
사용자가 입력한 영양소 섭취 정보와 생활습관, 인구사회학적 특성을 함께 반영하여 개인별 고콜레스테롤혈증 위험도를 예측할 수 있는 모델을 설계하는 것을 목표로 합니다.  
또한 예측 결과와 함께 영양소 섭취 패턴 및 주요 위험 요인을 직관적으로 제공할 수 있는 서비스 구현을 지향합니다.

## 주요 목표
- 영양소 섭취 정보 기반 입력 구조 설계
- 주요 영양소 섭취 패턴 지표 생성
- 생활습관 및 인구사회학적 요인을 통합한 학습 데이터셋 구축
- 고콜레스테롤혈증 위험도 예측 모델 설계 및 구현
