# 1차 프로젝트 자료집

## 프로젝트명
생성형 AI 기반 영양소 섭취 패턴 분석을 통한 이상지질혈증 위험도 예측

## 프로젝트 소개
본 프로젝트는 식생활 변화와 영양 불균형으로 증가하는 이상지질혈증 위험에 주목하여, 개인의 음식 섭취 정보와 영양소 섭취 패턴을 기반으로 위험도를 예측하는 서비스를 기획한 프로젝트입니다.  
사용자가 입력한 음식명 및 식사 내역을 바탕으로 영양소를 산출하고, 섭취 패턴과 생활습관·사회경제적 요인을 함께 반영하여 개인별 이상지질혈증 위험도를 예측할 수 있는 모델을 설계하는 것을 목표로 합니다.  
또한 예측 결과와 함께 주요 위험 요인을 설명하고, 맞춤형 식습관 개선 방향을 제시하는 서비스를 구현 방향으로 설정하였습니다.

## 주요 목표
- 개인별 영양소 섭취량 산출 체계 구축
- 주요 영양소 섭취 패턴 지표 생성
- 생활습관 및 인구사회학적 요인을 통합한 데이터셋 구축
- 이상지질혈증 위험도 예측 모델 설계
